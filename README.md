# 🎮 Jogo de Nim

O **Nim** é um jogo de estratégia para dois jogadores baseado em remover objetos de pilhas.

---

## 📜 Regras

- O jogo começa com várias pilhas (ex: `[3, 4, 5]`)
- Os jogadores jogam em turnos
- Em cada turno, o jogador deve:
  - escolher **uma pilha**
  - remover **uma ou mais unidades**

❗ Regras:
- Só pode mexer em **uma pilha por turno**
- Deve remover **pelo menos 1**
- Não pode remover mais do que existe

---

## 🏆 Objetivo

👉 Quem remover o **último objeto vence**
